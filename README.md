# Ring Gallery

**Ring Gallery** is a full-stack web application developed with **Java Spring Boot** (backend) and **React** (frontend). The platform enables users to submit their custom-designed rings along with detailed metadata and multiple images. Each submission is tied to a user’s email and receives a generated UUID, allowing for future edits.

---

## Features

- User-friendly form to submit ring designs with name and email
- Detailed ring metadata including size, material, and description
- Support for multiple images per ring submission (One-to-Many relationship)
- UUID-based system to allow users to edit their submissions later
- RESTful backend API for creating, retrieving, updating, and deleting entries
- Modern, responsive UI built with React

---

## Tech Stack

### Backend
- Java 17+
- Spring Boot
- Spring Data JPA
- MySql
- UUID generation

### Frontend
- React
- Axios for API requests
- React Router
- CSS framework (e.g., Tailwind CSS or Bootstrap - optional)
